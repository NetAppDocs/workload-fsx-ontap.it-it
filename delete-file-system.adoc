= Elimina un file system in NetApp Workload Factory
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Per eliminare un file system in NetApp Workload Factory, è necessario prima eliminare tutti i volumi, le VM di archiviazione o le relazioni di replicazione associati al file system.

.Fasi
. Accedere utilizzando uno dei link:https://docs.netapp.com/us-en/workload-setup-admin/console-experiences.html["esperienze di console"^].
. Dal menu Archiviazione, selezionare *FSx per ONTAP*.
. Da *FSx for ONTAP*, seleziona il menu delle azioni del file system FSx for ONTAP che desideri eliminare.
. Selezionare *Gestisci*.
. Nella scheda *Panoramica*, seleziona il menu azioni.
. Selezionare *Delete* (Elimina).
. Nella finestra di dialogo Elimina file system FSX per ONTAP, immettere il nome del file system FSX per ONTAP da eliminare.
. Selezionare *Elimina* per confermare.
