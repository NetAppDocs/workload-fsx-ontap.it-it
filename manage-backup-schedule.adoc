= Gestire la pianificazione di backup per un file system FSX per ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Gestisci la pianificazione del backup per un file system FSx for ONTAP in NetApp Workload Factory.

.Fasi
. Accedere utilizzando uno dei link:https://docs.netapp.com/us-en/workload-setup-admin/console-experiences.html["esperienze di console"^].
. Seleziona il menu image:workloads-menu-icon.png["L'icona del menu hamburger viene utilizzata per navigare tra carichi di lavoro quali storage, eda, ai, database, vmware e amministrazione."] e quindi seleziona *Archiviazione*.
. Dal menu Archiviazione, selezionare *FSx per ONTAP*.
. Da *FSx for ONTAP*, seleziona il menu delle azioni del file system per cui aggiornare la pianificazione del backup, quindi seleziona *Gestisci*.
. Nella sezione Informazioni, seleziona l'icona a forma di matita accanto a *Volume backups*. L'icona a forma di matita appare accanto alla freccia a discesa quando il puntatore del mouse si posiziona sulla riga *Volume backups*.
. Nella finestra di dialogo *Volume backups*, fornire le seguenti informazioni:
+
.. *Backup automatici giornalieri*: Consente di attivare o disattivare la funzione. Se si disattiva la funzione, selezionare *Applica*. Se si attiva la funzione, completare la procedura riportata di seguito.
.. *Periodo conservazione backup automatico*: Immettere il numero di giorni per la conservazione dei backup automatici.
.. *Finestra di backup automatico giornaliero*: Selezionare *Nessuna preferenza* (l'ora di inizio del backup giornaliero è selezionata per l'utente) o *selezionare l'ora di inizio per i backup giornalieri* e specificare l'ora di inizio.
.. *Finestra manutenzione settimanale*: Selezionare *Nessuna preferenza* (l'ora di inizio della finestra di manutenzione settimanale è selezionata) o *selezionare l'ora di inizio per la finestra manutenzione settimanale di 30 minuti* e specificare un'ora di inizio.


. Selezionare *Applica*.
